# `souroldgeezer-architecture`

`souroldgeezer-architecture` ships the architecture plugin for ArchiMate®
models, OEF XML, and the layout/runtime support that belongs with them.

## Skills

| Skill | What it covers | Notes |
|---|---|---|
| [architecture-design](../../souroldgeezer-architecture/skills/architecture-design/SKILL.md) | Enterprise, solution, and application architecture models in ArchiMate® 3.2 OEF XML | Includes build, extract, review, render, validate, repair, drift, and lookup workflows |

## Packaging notes

- Claude Code and Codex share the same marketplace entry for this plugin.
- The plugin keeps the architecture reference, runtime, schemas, fixtures, and
  procedures together so the skill can load its own support files locally.
- Users who previously used the design plugin for architecture work now install
  this plugin as well.
